BTC cracker against your own addresses from `OwnBTCAdress.txt`.

- Wallet 1 first — if funded, continue to Wallet 2
- Shows current key + address in real time
- On match → saves in `BitcoinFoundIt.txt`
- It functions only with 2 wallets in the .txt file no more or less further updates in future

- 
```powershell
python check_btc_api.py
```

Requires:
```powershell
pip install ecdsa base58

Tutorial
First download the file check_btc_api.py and the compared OwnBtcAdress.txt file store it on your desktop.
1. Open powershell as adminstrator copy & paste the command "pip install ecdsa base58" wihtout " it installs the requirements for the tool
2. Copy & paste in python check_btc_api.py this runs the tool
if you get a error you need to install python please just install it over the original site https://www.python.org if you installed open power shell again type in python check_btc_api.py good luck
